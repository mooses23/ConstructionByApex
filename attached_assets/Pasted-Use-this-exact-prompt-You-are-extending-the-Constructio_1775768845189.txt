Seed these sources

1. OhioBuys / State of Ohio procurement
Use OhioBuys as a primary Ohio state procurement source. Ohio DAS says OhioBuys is the State of Ohio’s online purchasing solution used by state agencies, boards, commissions, and the Ohio Attorney General’s Office, and the public contracts page supports filtering by keyword, commodities, contracting entity, supplier, and dates.

Seed source:
	•	key: ohiobuys_state
	•	name: OhioBuys - State of Ohio
	•	type: portal
	•	base_url: https://ohiobuys.ohio.gov
	•	geography: Ohio
	•	source_kind: state_procurement
	•	recommended_ingestion_method: search_review
	•	default_enabled: true

Also note that OFCC instructions tell users to access OhioBuys and browse public solicitations using filters like solicitation type and keyword.

2. Ohio Purchasing Group / BidNet Direct
Seed BidNet Direct’s Ohio Purchasing Group as a major aggregation source for Ohio state and local bids. BidNet says the Ohio Purchasing Group gives access to statewide and local government bids and RFPs, with open solicitations and keyword searching.

Seed source:
	•	key: bidnet_ohio
	•	name: BidNet Direct - Ohio Purchasing Group
	•	type: portal
	•	base_url: https://www.bidnetdirect.com/ohio
	•	geography: Ohio
	•	source_kind: state_local_bid_aggregator
	•	recommended_ingestion_method: search_review
	•	default_enabled: true

3. SAM.gov
Seed SAM.gov as a high-value source for federal opportunities located in Ohio or relevant to Ohio contractors.

Seed source:
	•	key: sam_gov
	•	name: SAM.gov
	•	type: api
	•	base_url: https://sam.gov
	•	geography: US / Ohio-filtered
	•	source_kind: federal_procurement
	•	recommended_ingestion_method: api
	•	default_enabled: true

4. Ohio Facilities Construction Commission related public opportunities
Seed OFCC-related opportunities as a priority class because Ohio law requires school district construction bids under OFCC projects to be publicly advertised. The Ohio Revised Code section on OFCC school construction says approved projects must advertise for construction bids.

Seed source:

	•	type: portal
	•	base_url: https://ofcc.ohio.gov
